# BalletDaily
BalletDaily is an app that connects with Apple Watch to sync movement and health data, and helps you log ballet sessions with enhanced notes for class names and instructors.

## Non-goals
- No social features
- No calorie burning / fitness gamification
- No coaching or teaching ballet
- No complex auth or user system

## Core Value
- Minimal user input
- Long-term trend visibility
- Calm, non-intrusive experience
